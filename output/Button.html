Button

Represents a button.

Primary

Use the primary modifier to create a primary button.

TWIG

{% include "c-button.html.twig" with {
    modifier: 'primary',
    url: '/',
    text: 'alle ansehen',
} %}

HTML

<a href="/" class="c-button c-button--primary" data-hide-on-print>
    <span class="c-button__title">alle ansehen</span>
</a>

RENDERED

alle ansehen
